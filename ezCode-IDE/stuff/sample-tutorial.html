EzCode Player

EzCode Player is a dynamic application that reads and plays EzCode packaged files, offering users a seamless and interactive experience to execute and visualize the code within. By intelligently interpreting the packaged project file, EzCode Player provides a comprehensive runtime environment, making it a valuable tool showcasing EzCode projects.

Top Bar

The Top Bar has the main navigation for the Editor.

File

Open: This opens the selected file.

Exit: This Exits the program.

Build

Play: This starts playing the project.

Stop: This stops playing the project.

Help

Toggle Show Errors: Toggles on and off showing any errors in the console.

The Textboxes below it show the installed versions of the Editor and the programming language.

Opened File Location

Shows the file location of the currently openned file.

Project List

Add Files Button: Adds files to the project list.

ListBox: This is a listbox that lets the user navigate between files easily. The List's contents can be saved to a file, and opened from a file.

Coding Box

This is where the user can type up there code.

Visual Output Box

This is where the output from the code is visualized.

Console Output

This is where the text ouput is seen.

Console Bar

The textbox that spans most of the bar is the console input. Whenever the Send button is pressed, or when enter is pressed, it will send the input to the script. The clear button clears the console.

Debbuger Space

This is where you can see values while the script is being debugged.
